🏮 The Lodge

Welcome, adventurer. Before you enter the fortress, read the ways of the realm.

📜 How This Works

1. Generate a Code — On the Admin screen, click "Generate Code". A secret code will appear.
2. Share the Code — Tell your players the code. They'll need it to enter the game.
3. Import on Player Screen — On the Player screen, type the code and click "Import Code".
4. Start the Game — Back on the Admin screen, click "Start Game". The 90-minute timer begins.
5. Solve Puzzles — Mark puzzles as solved when your players crack them. Doors unlock automatically.
6. Watch the Timer — Keep an eye on the countdown. When it reaches zero, the dragon wakes.

⚔️ Game Master Tips

💡 You have 5 hints to give. Use them wisely when players get stuck.

🔔 The Wake Dragon button simulates the dragon waking (scare effect). Calm it afterward.

🎬 Upload an intro video on the Admin screen, then click "Start Video" to play it on player screens.

🔗 Both screens must be on the same domain for pairing to work via localStorage.
