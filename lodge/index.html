Welcome, adventurer. Before you enter the fortress, read the ways of the realm.

📜 How This Works

1. Generate a Code — On the Admin screen, click "Generate Code".
2. Share the Code — Tell your players the code.
3. Import on Player Screen — Type the code, click "Import Code".
4. Start the Game — Click "Start Game". The 90-minute timer begins.
5. Solve Puzzles — Mark puzzles as solved. Doors unlock automatically.
6. Watch the Timer — When it reaches zero, the dragon wakes.
